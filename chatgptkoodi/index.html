Koodikielet

  • HTML: HTML (HyperText Markup Language) on standardoitu merkintäkieli, jota käytetään verkkosivujen rakentamiseen. Se määrittelee sivun rakenteen ja sisällön.
  • CSS: CSS (Cascading Style Sheets) on tyylikieli, joka mahdollistaa verkkosivujen ulkoasun ja ulkoasun muokkaamisen. Se määrittelee sivun visuaalisen ilmeen ja asettelun.
  • Python: Python on korkean tason ohjelmointikieli, joka on suosittu monipuolisuutensa ja selkeytensä ansiosta. Se sopii hyvin niin web-kehitykseen, tieteelliseen laskentaan, tekoälyyn kuin moniin muihinkin sovelluksiin.
